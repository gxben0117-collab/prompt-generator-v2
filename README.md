# 出圖自組咒語生產器 v2.0

真人電影級 AI 圖像生成提示詞產生器，專為 Claude Opus 4.7 設計。

## 🎯 核心理念

> **「真人演員被拍進奇幻世界」**

不生成 AI 美女，而是讓真人照片融入奇幻場景，保持原始臉部特徵與真實人體比例。

## ⚡ 快速開始

### 方法一：直接使用（推薦）
```bash
# 直接用瀏覽器打開
start index.html
```

### 方法二：本地伺服器
```bash
npm run dev
# 訪問 http://localhost:8080
```

## 📋 使用流程

1. 打開網頁
2. 選擇風格大類（魅魔、武俠、仙俠、古裝等）
3. 選擇場景主題
4. 填寫自訂欄位（主題、場景、服裝、妝容）
5. 點擊「產出咒語 + 複製咒語」
6. 在 ChatGPT 同一訊息中上傳照片並貼上咒語

## 🎨 核心功能

- **828+ 風格範例** - 涵蓋 22 個風格大類、396 個詳細分類
- **7 步驟咒語生成** - 鎖臉 → 身體比例 → 姿勢 → 場景服裝 → 鏡頭規格
- **13-tier 過濾系統** - 保護真人身份，防止 AI 換臉
- **電影攝影語言** - 將人類語言轉換成專業攝影術語
- **單頁應用** - 無需安裝，開箱即用

## 🛠️ 技術架構

### 7 步驟咒語生成順序
1. **鎖臉** - 真人身份優先
2. **身體比例** - 真實人體骨架
3. **姿勢** - 自然動作
4. **背景服飾道具** - 場景、服裝、道具
5. **鏡頭規格** - 焦距、角度、光線
6. **禁止咒語** - 13-tier 過濾
7. **最終覆蓋** - 身份主權強化

### 詞庫系統
- 主題詞庫（世界觀、身份）
- 場景詞庫（電影空間）
- 服裝詞庫（電影級服裝結構）
- 妝容詞庫（不改變骨相）
- 動作詞庫（攝影動作語言）
- 鏡頭詞庫（50mm 電影鏡頭）
- 光線詞庫（電影級光影）
