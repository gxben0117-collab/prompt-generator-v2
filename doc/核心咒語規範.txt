「真人演員真正穿上的電影級戲服」

而不是：

- 動漫 cosplay
- AI 插畫服裝
- 遊戲 skin
- 淘寶古裝

服裝需建立：

- 真實高級訂製服裝感
- 真實布料重量
- 真實人體穿著邏輯
- 真實電影服裝層次
- 真實金屬工藝
- 真實珠寶反光
- 真實空氣流動
- 真實垂墜結構

【世界觀自動延伸】

AI 可根據：

- 主題
- 朝代
- 神話
- 地域
- 民族
- 宗教
- 歷史
- 幻想宇宙
- 文明背景

自動延伸：

- 高級服裝結構
- 世界觀色彩
- 電影布料材質
- 珠寶工藝
- 金屬裝飾
- 頭飾吊飾
- 王族符號
- 宗教元素
- 地域文化辨識度
- 電影級輪廓語言

【服裝 Layer 系統】

Layer 1：
貼身基底／內襯

Layer 2：
胸腰支撐／馬甲結構

Layer 3：
主體服裝輪廓

Layer 4：
薄紗／披帛／外層布料

Layer 5：
腰封／護甲／金屬結構

Layer 6：
大型外袍／拖尾／主輪廓

Layer 7：
肩飾／羽飾／身份象徵

Layer 8：
珠寶／鏈條／吊飾

Layer 9：
頭飾／冠飾／文化配件

Layer 10：
最終電影輪廓與空氣感

【布料系統】

可自動建立：

- sheer silk chiffon
- translucent layered fabric
- heavy ceremonial silk
- velvet couture texture
- metallic embroidery
- reflective luxury textile
- handcrafted woven fabric
- royal brocade detailing
- lightweight spring gauze
- gravity-responsive drapery

布料需具有：

- 真實重力
- 自然垂墜
- 真實厚度
- 自然透光
- 真實反光
- 高級電影布料感

避免：

- 塑膠布料
- CG 材質
- 遊戲材質感
- 廉價 cosplay 感

【珠寶與頭飾系統】

- ornate ceremonial crown
- regional royal ornaments
- dangling gemstone tassels
- gold chain accessories
- jade pendants

- 多層金屬結構

避免：

【色彩電影系統】

依世界觀自動建立：

- 主色
- 副色
- 點綴色
- 光源色
- 地域色彩文化

例如：

- champagne gold
- warm ivory
- royal sapphire
- obsidian black
- crimson red
- jade green
- moonlit silver
- sunlit amber
- ruby red
- emerald green
- sapphire blue
- amethyst violet
- lantern gold
- peony pink
- cinnabar red

避免：

- 高飽和動漫色
- 螢光色
- 遊戲 UI 色彩
- 塑膠金屬色

不要把「高飽和動漫色」誤解成禁止高級艷麗。
允許真實電影燈光下的高級紅金、寶石色與絲綢反光。
