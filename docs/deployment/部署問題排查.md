## 🔍 問題排查

### 為什麼 Workflow 沒有自動運行？

可能的原因：
1. **Actions 未啟用** - 需要在倉庫設定中啟用
2. **權限問題** - 需要設定 workflow 權限
3. **首次推送** - 有時需要手動觸發第一次

### 如何檢查 Actions 是否啟用？

1. 前往：https://github.com/gxben0117-collab/prompt-generator-v2/settings
2. 左側選單找到 **Actions** → **General**
3. 確認 **Actions permissions** 設定為：
   - ✅ Allow all actions and reusable workflows
