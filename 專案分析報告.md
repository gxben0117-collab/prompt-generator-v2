# 出圖自組咒語生產器 - 完整專案分析報告

## 📋 1. 專案用途

這是一個**真人電影級 AI 圖像生成提示詞（Prompt）產生器**，專門為 Claude Opus 4.7 圖像生成功能設計。

### 核心功能
- **結構化輸入介面** - 透過表單收集用戶需求（主題、場景、服裝、妝容等）
- **智能咒語生成** - 將人類語言轉換成專業的電影攝影語言
- **真人身份保護** - 確保生成的圖像保持真人特徵，不被 AI 美化或替換
- **風格範例庫** - 內建 828+ 個電影級角色卡範例

### 設計理念
> **「真人演員被拍進奇幻世界」**

不是生成 AI 美女角色，而是讓真人照片融入奇幻場景，保持：
- 原始臉部特徵
- 真實人體比例
- 電影攝影感
- 專業服裝設計

## 🛠️ 2. 使用技術

### 前端技術
- **純 HTML/CSS/JavaScript** - 無框架依賴
- **單頁應用（SPA）** - 所有功能集成在 index.html（107KB）
- **內嵌數據** - CATS 陣列直接嵌入 HTML，無需外部 API

### 核心系統
1. **7 步驟咒語生成順序**
   - STEP 1: 鎖臉（真人身份優先）
   - STEP 2: 身體比例（真實人體骨架）
   - STEP 3: 姿勢（自然動作）
   - STEP 4-6: 背景服飾道具
   - STEP 7: 鏡頭規格（焦距、角度、光線）

2. **13-tier Prompt Governance**
   - 多層過濾系統
   - 防止 AI 換臉
   - 避免美貌模板污染

3. **詞庫系統**
   - 主題詞庫（世界觀、身份）
   - 場景詞庫（電影空間）
   - 服裝詞庫（電影級服裝結構）
   - 妝容詞庫（不改變骨相）
   - 動作詞庫（攝影動作語言）
   - 鏡頭詞庫（50mm 電影鏡頭）
   - 光線詞庫（電影級光影）

### 開發工具
- **Node.js** - 用於建置腳本
- **Python HTTP Server** - 本地開發伺服器
- **Git** - 版本控制

## 🚀 3. 如何啟動

### 方法一：直接打開（推薦）
```bash
# 直接用瀏覽器打開
start index.html
```

### 方法二：本地伺服器
```bash
# 啟動開發伺服器
npm run dev

# 或使用 Python
python -m http.server 8080

# 訪問 http://localhost:8080
```

### 使用流程
1. 打開網頁
2. 選擇風格大類（如：魅魔、武俠、仙俠）
3. 選擇場景主題
4. 填寫自訂欄位：
   - 主題（角色/職業）
   - 場景描述
   - 服裝描述
   - 妝容描述
   - 構圖/比例
   - 鏡頭角度
5. 點擊「產出咒語 + 複製咒語」
6. 在 ChatGPT 同一訊息中上傳照片並貼上咒語
